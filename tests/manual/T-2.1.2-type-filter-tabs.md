## M-2 筛选标签视觉区分

- **关联验收标准**: spec §验收标准（类型筛选标签 UI）

  1. 观察初始状态下标签外观
  2. 依次点击每个标签
  3. 观察 active 和非 active 标签的颜色差异
- ✅ **成功表现**（必须全部满足）:
  - active 标签有明显的蓝色填充（--primary 色）
  - 非 active 标签为透明背景 + 边框

  - hover 无任何视觉反馈

## M-3 Tab 键导航与 focus-visible

- **关联验收标准**: spec §边界条件（可访问性）
- **操作步骤**:
  1. 用鼠标点击搜索框，然后按 Tab 键
  2. 观察焦点是否移动到筛选标签上
  3. 用方向键或继续 Tab 在标签间切换
  4. 按 Shift+Tab 反向导航
- ✅ **成功表现**（必须全部满足）:
  - Tab 键能从搜索框导航到筛选标签
  - focus-visible 样式可见（蓝色 outline 环）
  - 标签之间的焦点顺序合理
- ❌ **失败表现**（出现任一即视为不通过）:
  - 键盘无法到达筛选标签
  - focus 状态下无任何视觉指示
  - 焦点顺序异常（如跳到页面其他区域后再回来）
